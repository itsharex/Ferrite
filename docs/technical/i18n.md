# Internationalization (i18n)

## Overview

Ferrite uses rust-i18n for internationalization, supporting multiple UI languages with runtime switching.

## Key Files

| File | Purpose |
|------|---------|
| `src/config/settings.rs` | `Language` enum, `language` field in Settings |
| `src/ui/settings.rs` | Language selector dropdown in Appearance section |
| `src/main.rs` | Apply saved language on startup via `set_locale()` |
| `locales/en.yaml` | English translations (430+ keys) |

## Implementation

### Language Enum

```rust
// src/config/settings.rs
#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize, Default)]
#[serde(rename_all = "lowercase")]
pub enum Language {
    #[default]
    #[serde(rename = "en")]
    English,
    // Future: ChineseSimplified, Japanese, etc.
}

impl Language {
    pub fn locale_code(&self) -> &'static str { "en" }
    pub fn native_name(&self) -> &'static str { "English" }
    pub fn all() -> &'static [Language] { &[Language::English] }
}
```

### Settings Integration

The `Settings` struct includes:
```rust
pub language: Language,
```

Default: `Language::English`

### Startup Language Application

```rust
// src/main.rs
let settings = load_config();
set_locale(settings.language.locale_code());
```

### Settings UI

Language selector in Settings → Appearance uses `egui::ComboBox`:
- Shows native language names with 🌐 icon
- Immediate application via `set_locale()`
- Auto-persisted with settings

## Translation Keys

Keys follow hierarchical naming: `category.subcategory.key`

Examples:
- `menu.file.open` - Menu items
- `settings.appearance.language` - Settings labels
- `dialog.confirm.ok` - Dialog buttons
- `status.words` - Status bar text

## Adding New Languages

1. Create locale file: `locales/zh-CN.yaml`
2. Add variant to `Language` enum:
   ```rust
   #[serde(rename = "zh-CN")]
   ChineseSimplified,
   ```
3. Update `locale_code()`, `native_name()`, `all()` methods
4. Add translations for all keys

## Dependencies

- `rust-i18n = "3"` - Core i18n framework
- `locales/` directory with YAML translation files

## Usage in Code

```rust
use rust_i18n::t;

// Simple key
let label = t!("menu.file.save");

// With parameters
let msg = t!("dialog.file_count", count = 5);
```

## Testing

- Missing keys fall back to English automatically
- All UI strings use `t!()` macro
- Language persists in `config.json` under `"language": "en"`
