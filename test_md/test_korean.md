# 한국어 단락 들여쓰기 테스트

이 파일은 한국어 단락 들여쓰기 기능을 테스트하기 위한 파일입니다. 설정에서 "Paragraph Indentation"을 "Chinese (2em)" 또는 "Japanese (1em)"으로 변경한 후 이 파일의 렌더링 결과를 확인해 주세요.

## 첫 번째 섹션: 기본 단락

이것은 첫 번째 단락입니다. 단락 들여쓰기가 활성화되면 이 줄의 시작 부분에 들여쓰기가 적용되어야 합니다. 한글 텍스트에서 들여쓰기는 일반적으로 2자(중국어 스타일) 또는 1자(일본어 스타일)입니다.

이것은 두 번째 단락입니다. 각 단락의 첫 줄에만 들여쓰기가 적용됩니다. 긴 텍스트가 여러 줄로 나뉘더라도 첫 줄만 들여쓰기됩니다.

이것은 세 번째 단락입니다. 모든 최상위 단락에 동일한 들여쓰기가 적용되어야 합니다.

## 두 번째 섹션: 인용문 내 단락

인용문 안의 단락은 들여쓰기가 적용되지 않아야 합니다:

> 이것은 인용문 안의 첫 번째 단락입니다. 인용문 내부의 텍스트에는 추가 들여쓰기가 필요하지 않습니다.
>
> 이것은 인용문 안의 두 번째 단락입니다. 여전히 들여쓰기가 없어야 합니다.

인용문 밖의 이 단락에는 다시 들여쓰기가 적용되어야 합니다.

## 세 번째 섹션: 목록과 단락

목록 항목:

- 첫 번째 항목
- 두 번째 항목
- 세 번째 항목

목록 다음의 이 단락에도 들여쓰기가 적용되어야 합니다. 목록 자체의 항목에는 들여쓰기가 적용되지 않습니다.

## 네 번째 섹션: 짧은 단락들

짧은 단락.

또 다른 짧은 단락.

마지막 짧은 단락.

## 다섯 번째 섹션: 혼합 콘텐츠

코드 블록 앞의 단락입니다:

```rust
fn main() {
    println!("안녕하세요!");
}
```

코드 블록 뒤의 단락입니다. 이 단락에도 들여쓰기가 적용되어야 합니다.

---

## 테스트 방법

1. Settings > Editor > Paragraph Indentation 으로 이동
2. "Off" 선택 → 들여쓰기 없음 확인
3. "Chinese (2em)" 선택 → 약 2글자 너비의 들여쓰기 확인
4. "Japanese (1em)" 선택 → 약 1글자 너비의 들여쓰기 확인
5. "Custom" 선택 → 원하는 값으로 들여쓰기 조절

각 설정에서 위 단락들의 첫 줄 들여쓰기가 올바르게 적용되는지 확인해 주세요.
